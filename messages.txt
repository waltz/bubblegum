
messages.txt

This file describes all of the messages that can be sent to/from any of the
bubblegum peers. All messages have a two byte identification code. Based on
that identification code, the peer knows how many other bytes to expect from
the socket.

===
GetPublicKey
===
A peer requests another peer's private key.
===
Code (2 Bytes): 00

===
PushPublicKey
===
A peer sends their private key to another peer.
===
Code (2 Bytes): 01
Public Key ()
